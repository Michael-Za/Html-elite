# Elite Partners - Landing Page

Premium landing page for Elite Partners US — featuring glassmorphism header, Three.js particle effects, Phosphor Icons, bento grid layout, and mobile-optimized horizontal scroll carousel.

## Deploy to Vercel

This is a static HTML site. To deploy:

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select this repository
4. Framework Preset: **Other**
5. Output Directory: `.` (root)
6. Click **Deploy**

Vercel will automatically serve `index.html` at the root URL.

## Tech Stack

- HTML5 / CSS3 (inline styles)
- Three.js (particle background)
- Phosphor Icons via Iconify
- Google Fonts (Clash Display, Cabinet Grotesk, Bebas Neue, Plus Jakarta Sans)
